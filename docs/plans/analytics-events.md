# Analytics Events (GA4)

## Event Names

- `lead_form_submit`
  - Trigger: successful quote/contact form submission
  - Params: `form_id`, `page_path`

- `call_click`
  - Trigger: click/tap on tel: links
  - Params: `phone_number`, `page_path`

- `cta_click`
  - Trigger: clicks on primary CTA buttons
  - Params: `cta_label`, `page_path`

## Notes
- Use consistent event names across the site.
