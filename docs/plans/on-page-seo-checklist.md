# On-Page SEO Checklist (Per Page)

- Title tag: primary keyword + city + brand
- Meta description: benefit + service + city + CTA
- One H1 that matches page intent
- H2s cover secondary keywords naturally
- Internal links to related services + contact
- Image alt text describes actual image content
- Schema: appropriate to page type (LocalBusiness/Service/FAQ)
- CTA above fold + repeated near bottom
- NAP consistent (name/address/phone)
- Performance: hero image optimized, lazy-load below fold
